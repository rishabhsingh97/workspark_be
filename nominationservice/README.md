# Nomination_service <hr />

## About <hr />
The Reward System is a platform designed to recognize and reward team members for outstanding performance based on nominations from team leads and decisions by a panel. The system will automate the manual process, ensuring transparency, efficiency, and accountability. The platform will also provide digitally signed certificates to rewarded users and maintain public links to access these certificates.

## Prerequisites
- Java 21
- Maven
- MySQL
- Spring Boot 3.4

## Instructions to run the project <hr />
Step 1: Build the project
```shell
mvn clean install
```

Step 2:
Run application `NominationServiceApplication`

## Configurable items<hr />
Team leads can nominate employees with justifications for their contributions and list and track nominations with real-time status updates.
